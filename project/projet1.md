## Description du Projet

Le projet Groupie Tracker consiste à créer une application web qui utilise des APIs pour suivre et afficher les informations sur les concerts de différents artistes musicaux.

## Fonctionnalités Principales

- Affichage des informations des artistes (nom, membres, date de création)
- Visualisation des dates et lieux de concerts
- Interface de recherche et de filtrage
- Carte interactive des concerts

## Technologies Utilisées

- Backend : Go (Golang)
- Frontend : HTML, CSS, JavaScript
- APIs RESTful
- Base de données pour le stockage des informations

## Objectifs du Projet

- Créer une interface utilisateur intuitive et responsive
- Implémenter une gestion efficace des données d'API
- Assurer une performance optimale de l'application
- Mettre en place un système de mise à jour en temps réel

## Structure du Projet

Le projet est organisé en plusieurs composants :

- API Handler - Gestion des requêtes API
- Interface Frontend - Design et interactions utilisateur
- Base de données - Stockage et gestion des données
- Services - Logique métier et traitement des données
